# shop1-nodejs

## Установка
```
npm install
```

### База данных PostgreSQL
Бэкап базы данных хранится в /server/database/DATABASE.sql

### Запуск в режиме разработки
```
npm run dev
```

### Компилирование для production
```
npm run build
```
